Define password validation rules

  • Min length: 8
  • Require uppercase: Yes
  • Require lowercase: Yes
  • Require numbers: Yes
  • Require symbols: Yes

Generate random salt and master key

  • Salt: 32-byte
  • Master key: 32-byte

Generate commitment to user's password

  • Password: MySecretPassword
  • Commitment: SHA-256 hash of salt + password

Generate key pair for password manager

  • Private key: x25519
  • Public key: Serialized and sent to user

User sends challenge to password manager

Password manager generates response to challenge

  • Response: Generated using function with inputs of challenge, salt, password rules, commitment, master key, and password manager's private key

Password manager sends response to user

User verifies response using password manager's public key

  • Inputs: Response, challenge, salt, password rules, and commitment

If response is valid, password is deemed valid
